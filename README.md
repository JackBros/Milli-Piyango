# Milli Piyango
Milli piyango, bir otomasyon uygulamasıdır. Bu uygulamada temsili olarak, 3 rakamlı rastgele piyango biletleri oluşturulmuştur. Bilet tipine göre alacakları ücretler belirlenmiştir.
